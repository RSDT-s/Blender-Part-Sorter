# Blender-Part-Sorter ESPAÑOL:
Este script permite que blender haga automaticamente colecciones de objetos, es principalmente util en el Modding de NFS, en el que muchas veces se puede llegar a tener un montón de objetos.

## Que hace exactamente

El script es simple pero efectivo, una vez activo, cada vez que tengas mas de 3 objetos con el mismo prefijo, blender creará una colección para estos objetos (en el Modding de NFS esto es extremadamente útil), además, si deseas sacar un determinado objeto de una colección, puedes hacerlo, el script no volverá a tocar este objeto hasta que tu mismo lo coloques nuevamente en su colección.

## Como se usa?

Por ahora no es necesario hacer nada para hacerlo funcional, todo se realiza automaticamente.

## Como se instala?

Una vez tengas Blender abierto, dirigete a Edit -> Preferences -> Addons, allí, pulsa el icono de "∨" -> Install from Disk, hecho esto busca Blender Part Sorter.py y añadelo a blender, listo, el srcript ya estará funcionando.

# Blender-Part-Sorter ENGLISH:
This script allows Blender to automatically create collections of objects. It is mainly useful in NFS modding, where you can often end up with a lot of objects.

## What exactly does it do?

The script is simple but effective. Once activated, every time you have more than three objects with the same prefix, Blender will create a collection for these objects (this is extremely useful in NFS modding). Furthermore, if you want to remove a specific object from a collection, you can do so, and the script will not touch this object again until you place it back in its collection yourself.

## How do you use it?

For now, you don't need to do anything to make it work; everything is done automatically.

## How do you install it?

Once you have Blender open, go to Edit -> Preferences -> Addons. There, click on the ‘∨’ icon -> Install from Disk. Once you have done this, search for Blender Part Sorter.py and add it to Blender. That's it! The script will now be working.
