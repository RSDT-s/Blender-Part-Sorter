# Blender-Part-Sorter
Este script permite que blender haga automaticamente colecciones de objetos, es principalmente util en el Modding de NFS, en el que muchas veces se puede llegar a tener un montón de objetos.

## Que hace exactamente

El script es simple pero efectivo, una vez activo, cada vez que tengas mas de 3 objetos con el mismo prefijo, blender creará una colección para estos objetos (en el Modding de NFS esto es extremadamente útil), además, si deseas sacar un determinado objeto de una colección, puedes hacerlo, el script no volverá a tocar este objeto hasta que tu mismo lo coloques nuevamente en su colección.

## Como se usa?

Por ahora no es necesario hacer nada para hacerlo funcional, todo se realiza automaticamente.

## Como se instala?

Una vez tengas Blender abierto, dirigete a Edit -> Preferences -> Addons, allí, pulsa el icono de "∨" -> Install from Disk, hecho esto busca Blender Part Sorter.py y añadelo a blender, listo, el srcript ya estará funcionando.
